Scope of Academic Exchange Project
1. Project Scope Overview
The scope of the Academic Exchange project defines the boundaries and capabilities of the system. The application focuses on providing a secure and user-friendly marketplace for students to exchange academic items while enabling administrative monitoring and control.

2. In-Scope Features
2.1 User Scope
User registration and authentication
Adding academic items with multiple images
Viewing all available products
Viewing own uploaded products
Location-based product discovery
Product detail view with image slider
Chat feature between users
Logout functionality
2.2 Admin Scope
Dedicated admin login page (not visible to users)
Admin dashboard with real-time data
View all registered users
View all uploaded products with images
Delete users or products if necessary
Monitor application usage
2.3 Technical Scope
Backend REST APIs for users, products, admin
SQLite database integration
Image upload and storage
Token-based authentication
Cross-device access using IP-based hosting
3. Out-of-Scope Features
The following features are not included in the current version:

Online payment gateway integration
Order tracking system
Email/SMS notifications
Mobile application (Android/iOS)
Product reviews and ratings
AI-based recommendations
4. Assumptions
Users have basic internet access
Images are uploaded in supported formats
Admin credentials are predefined
Application is used mainly within academic communities
5. Constraints
Uses local database (SQLite)
Limited to web-based access
No real-time payment handling
Image storage limited to server capacity
6. Future Scope
The project can be enhanced in future by:

Integrating online payment gateways
Developing a mobile application
Adding notification services
Implementing product ratings and reviews
Enhancing security with OTP/email verification
Hosting on cloud platforms for scalability
7. Final Note
The Academic Exchange project is designed with scalability in mind. While the current scope focuses on core academic exchange features, the architecture allows easy expansion to support advanced functionalities in the future.
